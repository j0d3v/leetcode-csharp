# LeetCode Solutions in C#

This repository contains my LeetCode solutions written in C#.

I'm using these exercises to refresh my knowledge of data structures and algorithms while getting more comfortable with modern C# and .NET. I try to solve problems on my own first, then revisit and optimize the solutions as I learn better approaches.

## Approach

I generally follow a simple test-driven workflow:

1. Write test cases from the problem requirements.
2. Implement a solution.
3. Refactor and optimize while keeping the tests passing.

## Structure

```text
Problems/
└── <ProblemName>
    ├── Solution.cs
    └── Tests.cs
```

* `Solution.cs` contains the implementation.
* `Tests.cs` contains a lightweight test harness and test cases.

I keep the testing setup intentionally simple and framework-free so I can focus on problem solving and maintain a fast feedback loop.

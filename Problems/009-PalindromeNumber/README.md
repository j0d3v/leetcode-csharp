# 009 - Palindrome Number

## Problem Description

Given an integer `x`, return `true` if `x` is a palindrome, and `false` otherwise.

A palindrome is a number that reads the same backward as forward. For example, `121` is a palindrome, but `123` is not.

### Examples:

1. **Input**: `x = 121`  
   **Output**: `true`  
   **Explanation**: `121` is a palindrome.

2. **Input**: `x = -121`  
   **Output**: `false`  
   **Explanation**: Negative numbers are not palindromes because of the negative sign.

3. **Input**: `x = 10`  
   **Output**: `false`  
   **Explanation**: `10` is not a palindrome.

4. **Input**: `x = 0`  
   **Output**: `true`  
   **Explanation**: `0` is a palindrome.

### Constraints:

- `-2^31 <= x <= 2^31 - 1`
